Informations

This website has been created to get bugs from users of in different web applications.
This website is an original idea from Cédric Jung. It should be host by the developer of those different web applications.
The goal is to provide a link to those web applictions which drives those users to this website.
When the users are on this website, they can report the bugs they have seen.
Now the developer can have a feedback from the users and improve his application.
